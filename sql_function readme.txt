## 使用方法及注意事项
### 运行环境
* 该程序使用python3版本编写，不兼容python2版本

### 使用方法
* 查询: select staff_id,name,age from staff_table where age > 22
* 修改: update staff_table set dept="IT" where name = "alex"
* 新增: insert into staff_table values (name,age,iphone,dept,enroll_date)
* 删除: delete from staff_table where staff_id = 4
